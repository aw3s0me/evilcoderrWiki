====== Типы данных ======
3 основныx типа данных:
  * text
  * number
  * boolean
====== Константы и переменные ======
**JS НЕ ПОЗВОЛЯЕТ ВЫРАЖАТЬ ТИПЫ ДАННЫХ ЯВНО**
===== Константы =====
    const CONSTNAME = value; //имя определяется автоматически, большие буквы облегчают нахождение ее
===== Переменные =====
    var varName = value; //неявный тип
===== Идентификация =====
Переменные и константы и другие конструкции определяются идентификаторами
  * Ид должен быть хотя бы длиной в 1 символ
  * Первый символ ид - _ или $
===== Особенности =====
    "1"+"2" = "12"
Поэтому нужно использовать методы parseInt() и parseFloat() для сложения чисел
    parseInt("1")+parseInt("2")=12 //при ошибке возвращает NaN
