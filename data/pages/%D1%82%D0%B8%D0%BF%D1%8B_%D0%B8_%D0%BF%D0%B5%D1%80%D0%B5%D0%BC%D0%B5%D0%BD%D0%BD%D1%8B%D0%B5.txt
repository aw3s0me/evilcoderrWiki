===== Названия переменных =====
Имена переменных могут состоять из следующих элементов:
  * букв латинского алфавита (заглавные и строчные буквы, опять же, различаются),
  * знаков подчеркивания
  * цифр (однако начинаться с цифры названия переменных не могут)
Правильные имена переменных:
  * x
  * y
  * luna2
  * kak_sebya_chuvstvuete

Неправильные имена переменных:
  * 2x
  * хьюстон у нас проблеммы
  * pribori otkazali
===== STRING =====
В string Можно использовать и двойные кавычки, и одинарные:
  * "laboratorka"
  * '2021'

Перед русскоязычными строчками надо ставить букву u (это значит используется кодирование unicode), иначе Python не поймет этих букв:
  * u"Прикольные картинки!!!"
===== Глобальные переменные =====
     x = 0
     def inc2():
        x = x + 1
        print x
Выдает ошибку, надо добавить дополнительное слово **global**:
     x = 0
     def inc2():
        global x
        x = x + 1
        print x
