===== Команды для выполнения лабы =====
  * mkdir - создание папки
  * touch file.txt - создание файла
  * vim file.txt - открыть на редактирование текст файл
  * :r !date, для того чтобы вставить в файл результат выполнения системной команды date. И :q выйти, :w выйти с сохранением
  * cp filename pathToCopy - сохранить через команду cp
  * ИЛИ для копирование F5
  * Зайти на FTP - левая панель или правая - FTP соединение
  * Распаковать tar.bz файл - tar -xzf filename
