====== Процедуры ======
Хранимая процедура - программа выполняющаяся внутри базы, к-ая принимает данные от пользователя и выполняет действия
===== Типы процедур =====
  - Системные хранимые процедуры (для адиминистративных действий)
  - Пользовательские хранимые процедуры (любые действия)
  - Временные хранимые  процедуры (временные потом уничтожаются)
  - Локальные временные хранимые процедуры (только в том соединении в к-ом созданы)
  - Глобальные временные хранимые процедуры (в любом)
===== Структура запроса =====
     <определение_процедуры>::=
     {CREATE| ALTER} PROC[EDURE]имя_процедуры
     [;номер]
     [{@имя_параметра тип_данных } [VARYING ]
     [=значение_по_умолчанию][OUTPUT] ][,...n]
     [WITH { RECOMPILE | ENCRYPTION | RECOMPILE,
     ENCRYPTION }]
     [FOR REPLICATION]
     AS
     Тело процедуры;
==== Обозначения ====
  * Номер  в  имени – это  идентификационный  номер хранимой процедуры
  * OUTPUT - параметр для возвращения данных
  * VARYING - значение по умолчанию
  * DEFAULT - параметр по умолчанию
  * RECOMPILE - для перекомпиляции при каждом вызове
  * ENCRYPTION - шифрование
===== Выполнение процедуры =====
    EXEC [ UTE]имя_процедуры [;номер]
    [[@имя_параметра=]{значение | @имя_переменной}
    [OUTPUT]|[DEFAULT]][,...n]
===== Пример процедуры =====
    CREATE Procedure ExamResults
    AS
    SELECT s.FIO AS 'ФИО', l.Nazvanie AS 'Дисциплина',
    p.Examen AS 'Оценка за экзамен'
    FROM Students AS s INNER JOIN
    Progress AS p ON p.ID_Student = s.ID_Student INNER JOIN
    Lessons AS l ON l.ID_Lesson = p.ID_Lesson
    WHERE s.FIO = 'Иванчуков А.Г.';
    /Для обращения к процедуреможно использовать команду
    EXECUTE ExamResults;
