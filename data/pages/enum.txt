====== Enum ======
  * Члены типа - константы
  * Им присваиваются числовые значения
  * IsDefined метод для проверки параметров

      if (!Enum.IsDefined(typeof(Color),c)) {
         throw...
      }
.
  * В основе типа числ значения
Пример:
     internal enum Color : byte {
        White,
        Red,
        ...
     } 
.
  * Для перечисления с битовыми флагами - юзаем параметр [Flags]
