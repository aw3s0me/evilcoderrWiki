====== Функции в JS ======
Функция - это переменная значение к-ой является ссылкой на ее тело
===== Ссылки на функции =====
**Ссылки на функции позволяют связать хорошо обработчик события с самим событием!**
    var myFunc = function (myAttr) {   };
вызов:
    myFunc(23);
===== Привязка к событию =====
**function(evt) {} - литерал функции для Func(32);**
    document.getElementById("elem").onclick = function(evt) { Func(32); };
