====== Null-совместимые значимые типы ======
Переменная значимого типа не может иметь зн-ие null
  * В базе данных могут содержаться значения в int = null => затруднения работы с базами
  * System.DateTime - не может быть null, зато java.util.Date может => если java приложение передаст инф-ию о дате и времени в CLR => ошибка
Класс System.Nullable<T>
Пример:
       Nullable<Int32> x=5;
       Nullable<Int32> y=null;
       Второй вариант исп-я оператор "?":
       Int32? x=5
       Int32? y=null
==== Оператор объединения nullable значений ====
Обозначается - "??"
Правила:
  * Если левый операнд!=null, оператор возвращает значение, если == null, то возвращается зн-ие первого операнда (поддержка ссылоч)
