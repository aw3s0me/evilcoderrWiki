====== Javascript ======
Шаги выполнения:
  - Пользователь вбивает URI
  - Сервер выбирает какую страницу отобразить
  - Сервер отдает HTML/CSS страницу
  - Браузер обрабатывает HTML/CSS и отображает ее
  - и также обрабатывает JS код с помощью интерпретатора Javascript, встроенного в браузер
После открытия страницы в браузере, сервер выходит из игры, поэтому далее выполняется JS -> JS - язык клиента
===== Запомнить! =====
  * При закрытии браузера при перезагрузке JS ликвидирует ВСЕ переменные!!
