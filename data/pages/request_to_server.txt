====== Заголовок 1-го уровня ======
  - Request for the page is sent to the web server
  - web server determines that .aspx file extension is registered with ASP.NET (if another ext -> ASP.NET is not involved)
  - If the first time a page requested, ASP.NET creates APP domain + COMPILES WEB PAGE CODE
  - .aspx acts like a miniature program (firing events)
  - ASP.NET asks control to render to HTML MARKUP
