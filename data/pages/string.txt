====== Строки ======
==== Char ====
  * char в Unicod'е
  * Для конвертации наследование от Convert
  * "\n"=Environment.NewLine
==== String ====
  * Для конкатенации и если много операций со строками - юзаем StringBuilder
  * @"C:\Bla\Bla" - объявление буквальной строки. ЮЗАЕТСЯ ДЛЯ REGEX и адресов файлов
  * САМОЕ ВАЖНО О СТРОКАХ - ОНИ НЕИЗМЕНЯЕМЫ (нельзя изменить размер итд)
  * Для сравнений строк юзаем Equals/Compare. CompareTo - с учетом региональных стандартов
  * Для производительности - хранить строки в хеш таблице - String.Intern
=== Форматирование строк ===
Наследование от IFormattable
  * Для форматирования можно юзать свою версию ToString
Пример:
       Decimal price = 123.54M;
       String s=price.ToString("C", new CultureInfo("vi-VN")); //перевод в формат вьетнама
       или
       String s=String.Format("On {0:D}, {1} is {2:E} years old.", new DateTime(2010,4,22,14,35,5), "Aidan", 7);
       //С,D,E - форматы
